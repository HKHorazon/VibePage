# schedule.json 轉換規則

## 來源

本學期使用兩份 PDF 交叉驗證，資料寫入同一個 `src/schedule.json`：

| 檔案 | 頁數 | 格式 |
|------|------|------|
| `data/1142多遊系教師課表.pdf` | 11頁，每頁一位老師 | 老師視角：橫軸=星期，縱軸=節次 |
| `data/班級課表.pdf` | 14頁，每頁一個班級 | 班級視角：橫軸=星期，縱軸=節次 |

**處理順序**：先從教師課表萃取全部資料，再用班級課表補漏或修正資料品質問題。

---

## 工具

- Python `pdfplumber`：提取 PDF 表格
- 自訂解析函式：拆解每格的課程、教師、教室、班級

---

## JSON 結構（src/schedule.json）

Flat array，每筆代表一個課程時段：

```json
[
  {
    "teacher": "楊國隆",
    "day": "一",
    "period": "2",
    "course": "專案管理",
    "room": "MB208",
    "class": "日四技多遊系2年乙班"
  }
]
```

| 欄位 | 說明 | 範例值 |
|------|------|--------|
| `teacher` | 老師姓名 | `楊國隆` |
| `day` | 星期（一～五） | `一` |
| `period` | 節次 | `1`～`8`、`A`～`D` |
| `course` | 課程名稱 | `專案管理`、`(*)基礎遊戲製作` |
| `room` | 教室代碼 | `MB208`、`G501`、`EB101` |
| `class` | 班級資訊 | `日四技多遊系2年乙班` |

## 教師課表解析規則（1142多遊系教師課表.pdf）

### 1. 節次對照

| PDF 原文 | JSON period |
|----------|-------------|
| 第1～第8 | `1`～`8` |
| 第A～第D | `A`～`D` |
| 中午 | 略過 |
| 第9、第10 | 略過 |

### 2. 星期對照

| PDF 原文 | JSON day |
|----------|----------|
| 星期一 | `一` |
| 星期二 | `二` |
| 星期三 | `三` |
| 星期四 | `四` |
| 星期五 | `五` |

### 3. 儲存格合併（join_lines）

PDF 提取的文字常在中文詞中間斷行，例如：
```
產品⾏
銷 M203
⽇四技多遊
系1年⼄班
```

合併規則：
- 中文接中文：直接相接，**不加空格**
- 英數接中文 / 中文接英數：直接相接
- 英數接英數：加一個空格

結果：`產品行銷 M203 日四技多遊系1年乙班`

### 4. 欄位拆分（parse_cell）

合併後文字，用教室代碼 regex `[A-Z]{1,5}\d{3,4}` 當分隔點：
- 教室代碼**前**的文字 = `course`
- 教室代碼本身 = `room`
- 教室代碼**後**的文字 = `class`

若無教室代碼：
- 嘗試找 `日四技` / `夜四技` 作為 class 起始點
- 若都找不到：整段當 course，room 和 class 為空字串

### 5. 略過的儲存格

- 空格或空白
- 含 `Office Hours` 的格子

## 班級課表解析規則（班級課表.pdf）

### 格式說明

每頁 header 包含班級名稱，例如 `班級:日四技多遊系2年甲班`。  
表格結構：

| 欄位 | 說明 |
|------|------|
| col 0 | 節次（`1`～`8`、`A`～`D`、中午、9、10） |
| col 1 | 時間（略過） |
| col 2 | 星期一 |
| col 3 | 星期二 |
| col 4 | 星期三 |
| col 5 | 星期四 |
| col 6 | 星期五 |
| col 7–8 | 星期六、日（略過） |

每個課程格子格式（以 `\n` 分隔）：
```
課程名稱（可能跨多行）
老師姓名
教室代碼
```

### 略過的節次

`中午`、`9`、`10`

### 欄位拆分（parse_cell）

1. 從最後一行往前找符合 `^[A-Z]{1,5}\d{3,4}$` 的行 → `room`
2. room 的前一行 → `teacher`
3. teacher 之前所有行 → `course`（用 join_cjk 合併）
4. `class` 取自頁面 header 的班級名稱

### 過濾規則

- 若 room 不存在（格子只有課程名稱、無教室），略過此格
- 若 teacher 不在 `src/teachers.json` 的 key 中，略過此格（不加入其他老師）

### 去重規則

以 `(teacher, day, period)` 為複合 key，已存在於 schedule.json 中的不重複寫入。

## 重新產生方式

```bash
# 教師課表 → schedule.json（初次建立）
python data/convert_teacher.py   # 或手動執行當時腳本

# 班級課表 → 補漏與修正（之後跑）
python data/convert_class.py     # 或手動執行當時腳本
```

> 新學期更新：替換兩份 PDF 後依序重跑兩個腳本即可覆蓋 `src/schedule.json`。
