## 實作規劃（已完成）

### 版面結構
- 最上方：標題按鈕列「多遊系課表 [模式名]」，點擊開啟搜尋蓋層
- 整頁一個課表格
  - 欄：期別標籤（窄）+ 週一～週五（5欄）
  - 列：第1、2、3、4、5、6、7、8、A、B、C、D（12列）
- 響應式策略：寬度優先填滿（`width: 100%`），格高自動由剩餘 viewport 高度計算
  - `--cell-h = max(16px, (100dvh - 134px) / 12)`
  - overhead 134px = body padding(16) + head-h(24) + 12 gaps(36) + 2 sep(12) + btn-mt(8) + btn-h(38)
  - 全尺寸單頁呈現，無捲動

### 搜尋蓋層（已完成）
點擊標題按鈕開啟全畫面蓋層，4 個分頁按鈕切換模式：
- **Tab 1 點擊格**：立即清空所有選擇，關閉蓋層，回到空白狀態
- **Tab 2 依老師**：顯示老師列表，點擊後高亮該老師所有課
- **Tab 3 依班級**：顯示班級列表，點擊後高亮該班級所有課
- **Tab 4 依教室**：顯示教室列表，點擊後高亮使用該教室的所有課

### 互動 — 搜尋模式（四種）

#### 模式一：點擊時間格（依星期/節次）
- 點擊格子後，高亮顯示該格（灰底 → 亮白底）
- 可點擊日期標頭、節次標頭、早/午/晚 批次選取（外圍模式）
- 高亮格子顯示老師縮寫；有課老師深色、無課老師灰色

#### 模式二：依老師搜尋
- 標題顯示「老師模式－[老師名]」
- 高亮顯示該老師所有有課的格子
- 格子內顯示：課程名 / 教室・班級（短名）

#### 模式三：依班級搜尋
- 標題顯示「班級模式－[班級短名]」
- 高亮顯示該班級所有有課的格子
- 格子內顯示：課程名 / 老師名・教室

### 模式四：依教室搜尋
- 標題顯示「教室模式－[教室]」
- 高亮顯示使用該教室的所有格子
- 格子內顯示：課程名 / 老師名・班級（短名）

### 搜尋邏輯（前端 filter）
資料為 flat array，所有搜尋都用 `Array.filter()`：
```js
// 依時間格 → 傳入 day + period
schedule.filter(s => s.day === day && s.period === period)

// 依老師
schedule.filter(s => s.teacher === name)

// 依班級（模糊比對）
schedule.filter(s => s.class.includes(keyword))
```
