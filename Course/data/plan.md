## 實作規劃（待確認）

### 版面結構
- 無標題
- 整頁一個課表格
  - 欄：期別標籤（窄）+ 週一～週五（5欄）
  - 列：第1、2、3、4、5、6、7、8、A、B、C、D（12列）
- 手機直式：全寬自適應，每格最小 44px 觸控區域

### 互動 — 搜尋模式（三種）

#### 模式一：點擊時間格（依星期/節次）
- 點擊格子後，高亮顯示該格（灰底 → 亮白底）
- 同時在格子下方或側邊列出「此時段有哪些老師在上課」
- 顯示：老師、課程名、教室、班級

#### 模式二：依老師搜尋
- 頁面頂部有老師選單（下拉或按鈕列）
- 選擇老師後，高亮顯示該老師所有有課的格子
- 點擊高亮格子可查看該節課程細節

#### 模式三：依班級搜尋
- 輸入關鍵字（如「1年甲班」「2年乙班」「夜四技」）
- 高亮所有符合的格子
- 多個老師可能同時高亮（同一班級有多堂課）

### 搜尋邏輯（前端 filter）
資料為 flat array，所有搜尋都用 `Array.filter()`：
```js
// 依時間格 → 傳入 day + period
schedule.filter(s => s.day === day && s.period === period)

// 依老師
schedule.filter(s => s.teacher === name)

// 依班級（模糊比對）
schedule.filter(s => s.class.includes(keyword))
```

### 視覺風格
| 元素 | 樣式 |
|------|------|
| 頁面背景 | `#111`（深黑） |
| 未選格背景 | `#2a2a2a`（深灰） |
| 格線 | `1px solid #3a3a3a` |
| 選中/高亮格 | 白底 `#ffffff` + 深色文字 |
| 期別標籤 | 小字、低對比（`#666`） |
| 日期標頭 | 小字、中對比（`#999`） |
| 字體 | 系統預設，無外部字體 |
